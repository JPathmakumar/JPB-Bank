⚙ Supabase Setup Required:

1. Go to supabase.com → New project
2. Go to SQL Editor and run:
CREATE TABLE accounts ( id bigserial PRIMARY KEY, name text NOT NULL, account_number text, balance numeric DEFAULT 0, role text DEFAULT 'client', password text, created_at timestamptz DEFAULT now() ); CREATE TABLE transactions ( id bigserial PRIMARY KEY, account_id bigint REFERENCES accounts(id) ON DELETE CASCADE, type text, description text, amount numeric, created_at timestamptz DEFAULT now() ); ALTER TABLE accounts ENABLE ROW LEVEL SECURITY; ALTER TABLE transactions ENABLE ROW LEVEL SECURITY; CREATE POLICY "open" ON accounts FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "open" ON transactions FOR ALL USING (true) WITH CHECK (true); 3. Copy Project URL + anon public key from Settings → API
4. Paste them into SUPABASE_URL and SUPABASE_KEY at the top of this file
